## **3. 테스트 실행 및 결과 기록**
- 테스트 실행 후 결과값(`✅ / ❌`)을 업데이트하며, 실패한 경우 원인을 추가로 기록한다.
- 예상과 다르게 동작하는 경우, 버그 티켓을 생성하여 개발팀에 공유한다.
